## :dart: About ##

Python ile FastApi frameworku ile SSL sertifikası kullanarak giriş seviyede bir çalışma. Json veri trafiğini işlemek için ileride yapılacak çalışmalarda kullanılabilir.


## :rocket: Technologies ##

The following tools were used in this project:

- [Python](https://www.python.org/)
- [FastaPi](https://github.com/fastapi/fastapi)
- [uv](https://docs.astral.sh/uv/)


## :white_check_mark: Requirements ##

Before starting :checkered_flag:, uv ile kolayca skurulumları yapabilirsiniz.

## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/metin02oktay/FastApi_giris

# Access
$ cd fast_api_hello_world

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
